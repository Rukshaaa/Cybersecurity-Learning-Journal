# Lab 01 - Basic LAN Connectivity

## Objective
Connect two PCs using a switch and verify network communication.

## Devices
- 2 PCs
- 1 Switch (2960)

## IP Configuration
PC1: 192.168.1.10  
PC2: 192.168.1.11  

## Result
Ping successful between both PCs.

Sent = 4, Received = 4, Lost = 0

## Key Learning
- Switch connects devices in the same LAN
- Devices must be in same subnet to communicate
- ICMP ping is used to test connectivity
